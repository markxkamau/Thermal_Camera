# Thermal_Camera
Security Improvement using AMG8833-Arduino Thermal Imaging System.
Requirements 
1. AMG88xxx Camera
2. Aruino Board
3. ESP32-Cam
4. Buzzer
5. Jumper Cables
6. Resistors
